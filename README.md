# React + TypeScript + Vite

A simple React/TypeScript frontend app to serve small, low-latency models with edge deployment on supported browsers.  
Hosted on GitHub Pages [HERE](https://wapet99.github.io/browser-deployed-models/).

## Included models
### Driver Behaviour Classifier
A simple custom CNN classifier trained on the State Farm Distracted Driver dataset.  
PyTorch model exported to ONNX, deployed to browser using ONNX Runtime.
